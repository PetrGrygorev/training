### Что такое Git?

> *Git — это программа с текстовым интерфейсом, в которой лучше работать в командной строке.
Командная строка имеет все реализованные возможности по работе с Git.
Необходима программисту на компьютере
для работы с GitHub.*
>> __GitHub — это облачный хостинг репозиториев Git, или папок, где Git отслеживает изменения.
GitHub для программиста это прежде всего хранилище файлов кода.
Эта платформа также предлагает безопасную авторизацию пользователям,
инструменты тестирования и анализа кода, сервисы развёртывания проектов — GitHub Actions.__

Большинство современных редакторов кода имеют встроенный Git-клиент,
который позволяет взаимодействовать с Git с помощью графического интерфейса пользователя.
Но к сожалению графические редакторы реализуют не все полностью
возможности работы с Git. Каждый редактор имеет только те возможности,
которые посчитали нужными заложить разработчики данного графического редактора кода.
Поэтому лучше сразу приучиться работать с Командной строкой. После этого будет
намного легче понимать работу любого редактора с Графическим интерфейсом пользователя.
Это позволит вам использовать Git на любом компьютере,
где отсутствуют графические
редакторы для работы с Git. Например, работать с Git на удаленном сервере,
который не имеет графического пользовательского интерфейса.
